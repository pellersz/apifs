apifs commands and options:

start: - guess fucking what
stop: guess fucking what 2 electric boogaloo

notify: create notification
    once:
        -w (when): datetime Y-m-d H:M:S
        -p (picture): string
        -s (sound): string
        --desc (description): string
    daily:
        -w (when): time H:M:S
        -d (days): numbers 1 - 7, can use n-m syntax too (ex. 14526 or 1-6) 1 is monday
        -p, -s, --desc works the same
